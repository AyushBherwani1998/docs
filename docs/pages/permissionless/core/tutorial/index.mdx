# Tutorials

Learning-oriented lessons that take you through a series of steps to complete a project. Most useful when you want to get started with Pimlico.

[Tutorial 1](/permissionless/core/tutorial/tutorial-1) takes you through the journey of leveraging permissionless.js's high-level APIs to easily create and bundle a user operation.

[Tutorial 2](/permissionless/core/tutorial/tutorial-2) takes you through the journey of sponsoring your first user operation with USDC with an ERC-20 paymaster.
